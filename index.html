WHY CHOOSE ME?

My approach to programming is guided by the principles of SERM: Scalability, Efficiency, Reliability, and Modularity.

• Scalability

I ensure my code is long-term reliable by incorporating flexibility into my systems, allowing future developers to easily build upon existing work.

• Efficiency

I follow up-to-date best practices to write efficient code, minimizing unnecessary logic and optimizing algorithms.

• Reliability

Efficiency goes hand-in-hand with reliability. I ensure my code is dependable by incorporating thorough debugging and error-handling within my framework.

• Modular

I make my code modular, breaking down functions into sub-modules for easier navigation and maintenance by future developers.

I enjoy collaborating with other developers to create innovative solutions.

MY WORK

I thrive on developing new and innovative systems. Here's a glimpse into some of my technical achievements:

Modular Framework Design:

Frustrated with scattered content, I developed a framework to streamline development for myself and future developers. The initial version was a personal project, while the second version showcases significant improvements, particularly in its networking module. This module allows for easy access to remote events, and creating a base system is simple with a few lines of initiation code: 

local MyModule = {}

function MyModule.initiate()
    print("Module initiated!")
end

return MyModule

This modular approach not only enhances organization but also ensures that the codebase remains flexible and maintainable, making it easier for future developers to understand and build upon.

3D Chunk System Design:

This system generates an infinite plane around the player's character or camera. Built on Object-Oriented Programming (OOP) principles, it offers various methods, such as :SetAxis(axis, state) to disable specific axes and :SetActive(state) to control the chunk system.

Number Formatting:

My number abbreviator/formatter module displays base 10 numbers in various formats, including time formats. It’s a small but powerful tool showcased in the video on the left.

Pathlink Module:

This module creates customizable paths between points, useful for tutorials or map guidance. It supports style changes and custom marker points.
